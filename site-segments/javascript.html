JavaScript

this keyword

  • Arrow function: defined by the lexical environment of the arrow function
  • Function call on object: refers to the current object, in other cases the window object is referenced

Prototypes

  • A template object in which other objects may stem from. When an object's parent has a prototype this is referred to as a prototype chain
  • Properties and methods are bound to the prototype and not the object instance
  • Properties and methods are inherited if and only if they follow the pattern of Object.prototype
  • Every prototype has a constructor which points to the original constructor.

Classes

  • Utilizes prototype inheritance
  • Body of class is bound by strict mode(turns former mistakes in errors)

Immediate Invoked Functions

Closures

  • Lexical scoping: inner function having access to variables of the outer function
  • Closure: Variable assigned to a functions inner function which maintains lexical scoping
  • Have access to Local, Global, and outer function scoping
  • In a for loop with inner closures, the let keyword binds each closure to the current block, without this each closure in the loop will have the same lexical environment
  • Use closures wisely as they are costly with regards to performance

Module Pattern

  • Utilizing closures to ensure privacy of data
  • Module: small unit of reusable, maintainable, non nampespace polluting code
