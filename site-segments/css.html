Box Model

  • Every element in HTML is represent by a box
  • Consists of three layers: padding, border, & margin
  • Padding - Spacing separating content from the border
  • Border - Spacing that surrounds the content/padding and separates it from the margin
  • Margin - Spacing that separates elements from one another

Specificity

  • Determines how styles are applied to elements
  • Higher specificity overrules styling
  • If two styles are equal, in specificity, the later is selected
  • Inherited/universal styles have a specificity of 0
  • Broken into 4 levels
    • 1s digit - element, pseudo element
    • 10s digit - attribute, class
    • 100s digit - ID
    • 1000s digit - inline
