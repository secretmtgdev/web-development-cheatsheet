Common Properties

Display

  • Impacts the layout flow of an element
  • Values:
    • block: Take up full width by default and height/width can be set
    • inline: Wrap around the inner content, can't set height/width
    • inline-block: Can appear on the same line as other elements and can set height/width
    • none: Element doesn't appear on page and no space is allocated for it

Width & Height

  • Used to determine the size of a blockish element
  • min-height: Sets the smallest height that the element should be
  • max-height: Sets the tallest that an element can get
  • min-width: Sets the smallest width that the element should be
  • max-width: Sets the widest that an element can get
  • Values:
    • px: Uses monitor pixels for displaying
    • em: Based off of the parents font size of 'm'
    • rem: Based off of the root font size of 'm'
    • %: Bases off a certain size relative to the parent

Margin

  • Spacing of elements from one another
  • margin: 0 auto; - Center a block with respect to its parent

Padding

  • Space an elements content from its border

Border

  • Surrounding porition of an elements content separating the padding and the margin
  • Properties
    • border-width: How thick the border should be
    • border-style: Styling of the border
    • border-color: The color of the border

Float

  • Removed from the normal flow a page
  • Inline elements will wrap around it
  • Typically used for web layouts

Clearing Floats

  • Some floats don't fill the same heigh as other ones causing non float elements to be awkwardly pushed upward
  • This can be fixed with a clear float on both sides
  • Sometimes a container of divs collapses if the inner divs are floats, this can be solved by using the after pseudo selector, setting content to empty, display as a block, and clearing both sides of floats

Color

  • hex: #xxyyzz, first tuple maps to red, second to green, and third to blue
  • rgb: rgb(xxx, yyy, zzz), range is from 0 to 256 for each color
  • rgba: rgba(xxx, yyy, zzz, 0-1), same as rgb with an additional value for opaqueness

Background

  • Can set to a color
  • Can set to an image
  • Can set position relative to an element

Font

  • Can set the font style, height, and size with em, rem, px
  • Can set the variant to all caps or lowercase
  • Can set the height of each line

    • Box Model

    • Every element in HTML is represent by a box
    • Consists of three layers: padding, border, & margin
    • Padding - Spacing separating content from the border
    • Border - Spacing that surrounds the content/padding and separates it from the margin
    • Margin - Spacing that separates elements from one another

    Specificity

    • Determines how styles are applied to elements
    • Higher specificity overrules styling
    • If two styles are equal, in specificity, the later is selected
    • Inherited/universal styles have a specificity of 0
    • Broken into 4 levels
      • 1s digit - element, pseudo element
      • 10s digit - attribute, class
      • 100s digit - ID
      • 1000s digit - inline
